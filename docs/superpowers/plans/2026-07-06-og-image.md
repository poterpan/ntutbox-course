# course-web OG 圖（HTML 源稿 → 靜態 PNG）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 產出一張 1200×630 的全站共用 OG 圖（HTML 源稿進 repo、截圖成 `public/og.png`）並接進 metadata。

**Architecture:** `apps/web/og/og-image.html`（自包含 1200×630 單頁）→ headless Chrome `--screenshot` 輸出 `apps/web/public/og.png` → `layout.tsx` metadata 補 `openGraph.images`/`twitter.images`（metadataBase 已設，相對路徑即可）。

**Tech Stack:** 純 HTML/CSS（無依賴）、headless Chrome CLI 截圖、Next.js Metadata API。

## Global Constraints

- 視覺對齊網站：背景用 `globals.css` 既有漸層（radial 藍/紫/青 + linear 淡藍），accent `#3b82f6`、ink `#141a24`、衝堂橘 `#ea580c` 系；圓角級距照 `apps/web/AGENTS.md`。
- 圖上**無動態內容**（課名靠 og:title）；文案初稿「查課・排週課表・衝堂學分即時檢查」，以使用者視覺定稿為準。
- PNG 目標 **<300KB**；過大時先試 1200×630@1x，再不行改 JPEG（`og.jpg`，metadata 同步改）。
- **視覺定稿 = 使用者看截圖說 OK**；merge 前不算完成。
- 指令在 `apps/web/` 下執行。

### Task 2: 截圖 → `public/og.png`、視覺迭代到使用者定稿

**Files:**
- Create: `apps/web/public/og.png`

- [ ] **Step 1: 截圖**

```bash
cd apps/web/og
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
  --headless=new --screenshot="$(pwd)/../public/og.png" \
  --window-size=1200,630 --hide-scrollbars \
  "file://$(pwd)/og-image.html"
ls -lh ../public/og.png
```
Expected: 產出 `public/og.png`，尺寸 1200×630（`sips -g pixelWidth -g pixelHeight ../public/og.png` 驗證），檔案 <300KB（超標 → 改 JPEG 流程）。

- [ ] **Step 2: 自檢 + 給使用者看**

Read 該 PNG 目視（構圖、字距、對比、icon 邊緣），修 HTML 重截到自己滿意，再貼給使用者定稿。**使用者可能迭代數輪——每輪改 `og-image.html` → 重截 → 再看。**

- [ ] **Step 3: 定稿後 Commit**

```bash
git add apps/web/public/og.png
git commit -m "feat(web): #36 產出 og.png（1200×630）"
```

### Task 3: metadata 接線 + 驗證

**Files:**
- Modify: `apps/web/src/app/layout.tsx`

- [ ] **Step 1: 補 images**

`layout.tsx` 現有 metadata（metadataBase 已設 `https://course.ntutbox.com`）：

`openGraph` 物件內加：
```ts
    images: [{ url: "/og.png", width: 1200, height: 630, alt: "北科盒子 排課" }],
```
`twitter` 物件內加：
```ts
    images: ["/og.png"],
```

- [ ] **Step 2: build + 驗 tag**

```bash
cd apps/web && npm run lint && npx tsc --noEmit && npm test && npm run build
grep -oE '<meta (property="og:image"|name="twitter:image")[^>]*>' out/index.html
```
Expected: 全綠；兩個 image tag 存在且為絕對網址 `https://course.ntutbox.com/og.png`。

- [ ] **Step 3: Commit**

```bash
git add apps/web/src/app/layout.tsx
git commit -m "feat(web): #36 metadata 接 og:image / twitter:image"
```
