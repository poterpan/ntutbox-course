## Live 探測結論（2026-06-13，本計畫的事實基礎；fixtures 已存 `crawler/tests/fixtures/`）

1. **`stime` 為必要參數**：`QueryCourse.jsp` POST 缺 `stime` → 回「查詢選課資料出現錯誤」。全時段查詢 `stime="0"`（頁面 JS：全勾≥98 格時設 "0"）。
2. **「全校＋所有系所會被擋」已推翻**：那是前端 JS alert；server 端 `matric=全校13碼 + unit=＊` 正常回 2.2MB。→ 不過我們仍按系所分頁（要 unit 歸屬 + 回應較小）。
3. **`matric` 值格式**＝字面引號列表字串，如 `'1','5','6','7','8','9'`（日間部）；單一碼如 `'5'` 可獨查。全部 13 個單一碼：`0,1,4,5,6,7,8,9,A,C,D,E,F`。
4. **`unit=＊`**（全形星號）＝所有系所。
5. **課程編碼**（course_code，跨學期固定）在列內課名的 `Curr.jsp?format=-2&code=XXX` 連結 → **無需逐課請求 Curr.jsp**。
6. **節次牆鐘時間**在每個 QueryCourse 回應頁尾（官方）：1=08:10-09:00, 2=09:10-10:00, 3=10:10-11:00, 4=11:10-12:00, N=12:10-13:00, 5=13:10-14:00, 6=14:10-15:00, 7=15:10-16:00, 8=16:10-17:00, 9=17:10-18:00, A=18:30-19:20, B=19:20-20:10, C=20:20-21:10, D=21:10-22:00。
7. **`Subj.jsp?format=-2&year&sem`** → 每學期系所清單（114-1 共 61 個，含 教務處/體育室/通識中心/AA 校院級）；**`format=-3&code=系所碼`** → 該系班級清單（`format=-4&code=班級碼` 連結，班級碼=cwish cunum 命名空間）。
8. **表格 24 欄**、表頭：`課號|課程名稱|階段|學分|時數|修|班級|教師|日|一|二|三|四|五|六|教室|人|撤|授課語言|教學大綱與進度表|備註|隨班附讀|實驗實習|跨領域`。未閉合 `<td>` tag soup → html5lib。空白格是全形空白 `　`。
9. **year 下拉已含 115（預設）** → 115-1 應已公布；爬不到時警告並產空檔。

### 每學期爬取策略（≈75 請求/學期）
- `Subj.jsp -2` → 系所清單（1 請求）
- 每系所 `Subj.jsp -3` → classes.json 原料（~61 請求）
- 每系所 `QueryCourse(matric=全校13碼, unit=系所)` → 課程列 + unit 歸屬（~61 請求）
- 13 × `QueryCourse(matric=單一碼, unit=＊)` → `offering_id → 學制碼集合` → `division_group`（13 請求；同時當 per-dept union 的完整性 cross-check）
- 合併去重 by `offering_id`；只出現在 matric 查詢、沒被任何系所收的課 → unit=None + 警告

### 學制碼 → DivisionGroup 對映
`5,6,7`→day；`8,9,A,C,D`→graduate；`0,4,F`→extension；`1,E`→program；未知→other。原始碼集合存 `raw_fields["matric_codes"]`。
